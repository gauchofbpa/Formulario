HTML para explicar formulários e suas especificações.

Por: João Constantino Caetano

Os formulários servem para coletar algum dado do usuário.

O elemento form, é um elemento de container cuja função é configurar a forma como o formulário se comporta, podendo conter vários elementos específicos para isso, como input, label, select, button e etc.




O input type="text" define um campo para aceitar texto.

A tag "label" serve como um "rótulo" do input, identificando ao usuário o que é o respectivo campo.






O input type="password" serve para ocultar os caracteres no campo.

O input type="submit" quando clicado, envia os dados para serem processados.

OBS: Para isso acontecer, é necessário estar configurado com o action e method.

O form action="url" define o local em que os dados vão ser enviados.

O method="post/get" define qual o método HTTP vai ser usado para os dados.

O post vai basicamente enviar os dados, e o get vai pegar.




O input type="email" é um campo dedicado para e-mail, e em alguns navegadores, aparece os seus e-mails para enviar.

O input type="reset" faz com que todos os campos do form se apaguem.




O input type="checkbox" deixa você marcar várias opções.




O input type="radio" deixa você marcar somente uma opção.
